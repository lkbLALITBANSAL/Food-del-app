# 🍽️ EatZo – Food Delivery Web App

A full-stack **MERN food delivery application** where users can browse dishes, add items to cart, place orders, and pay securely using **Stripe**.
The project also includes an **Admin Panel** to manage food items and orders.

## 📌 Features

* Browse food items
* Add / remove items from cart
* Floating cart indicator
* Secure login with JWT authentication
* Stripe payment integration
* Order verification system
* View order history
* Admin dashboard for managing food items and orders

---

## 🧱 Tech Stack

### Frontend

* React.js
* Vite
* CSS
* React Router
* Axios

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose

### Authentication

* JWT (JSON Web Token)

### Payment

* Stripe

### Deployment

* Frontend: Vercel
* Admin Panel: Vercel
* Backend: Render
* Database: MongoDB Atlas
